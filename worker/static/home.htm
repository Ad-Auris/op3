The Open Podcast Prefix Project is a free and open-source podcast prefix analytics service committed to open data and listener privacy.
Podcasters or podcast hosting companies can prepend
${origin}/e/
to podcast episode urls in their feed to participate and start sending us data.
(No need to remove existing prefixes, there can be more than one)
This service is a departure from all other analytics services in that it is an open and auditable machine. Not only is the code available on GitHub, but all deployments are triggered there as well.
It runs on a global CDN with edge nodes in over 275 cites in over 100 countries.
This project is still very much in early development, but is ready to receive real data from early adopters and make it available over the OP3 API.
Learn more about our future plans on the GitHub readme.
